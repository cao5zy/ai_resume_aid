# 【人文季-公益单元投稿】AI微光求职——面向弱势就业群体的免费简历优化交互工具
## 一、项目核心理念
本次项目紧扣活动主题「用AI重新看见人」，聚焦残障人士、大龄再就业人群、乡村务工青年、偏远地区贫困应届生四类求职弱势群体的就业困境。市面上常规AI简历工具仅服务普通白领，门槛高、话术专业化，忽略弱势群体缺乏职场表达能力、存在求职自卑、信息获取闭塞等现实问题。本项目依托Vibe Coding搭建轻量化交互系统，以AI技术作为帮扶载体，消解就业不平等，践行AI向善的人文内核。

## 二、核心功能设计
1. **分人群定制化简历优化引擎**
针对不同群体调整优化逻辑：为残障求职者弱化身体局限、突出技能优势；给45岁以上大龄劳动者规避年龄歧视话术，重点凸显实操经验；面向低学历务工者使用通俗口语化引导，自动梳理零散工作履历；帮扶贫困应届生免费适配基层、公益、乡村振兴类岗位。
2. **无障碍交互适配**
适配屏幕朗读、大字体模式、语音输入，视障、肢体不便人群无需复杂打字即可完成简历填写与优化；支持多地方言语音交互，降低文字操作门槛。
3. **配套求职心理疏导AI模块**
弱势群体求职易产生自我否定、焦虑情绪，工具内置轻量化AI对话助手，在简历生成后提供情绪安抚、面试鼓励话术，同步推送公平就业维权科普内容。
4. **完全免费公益属性**
无会员、无付费解锁功能，不投放求职广告，面向目标群体永久开放使用；接入知乎搜索API，自动匹配适配弱势群体的包容性岗位资讯。

## 三、人文价值说明
AI可以快速润色文字、匹配岗位，但无法天然关照人的处境。本项目跳出“效率工具”的单一定位，把技术重心放在被就业市场忽视的人群身上：
许多残障、中老年、基层劳动者拥有扎实实操能力，却不懂包装自身经历，在求职阶段就被筛选淘汰。本工具用降低门槛的AI交互，弥补信息差与表达短板，让每一类劳动者的个人价值被完整看见。
呼应活动主旨：算力只是载体，关怀人的需求、弥合社会公平缺口，才是AI创作真正的人文意义。
